# Python version management

- Uses [`uv`](https://github.com/astral-sh/uv) to install/manage Python versions, dependencies, virtualenvs, and tools.
- Replaces `pyenv`, `pip`, `poetry`, `pipx`, and `virtualenv` with a single tool.

This is similar to using `nvm` for Node versions and `pnpm` for dependencies — but unified into one tool.
